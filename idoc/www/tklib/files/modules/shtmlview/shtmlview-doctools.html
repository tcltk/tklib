Name

shtmlview_doctools - shtmlview extension adding support for doctools markup

Synopsis

  • package require dtplite
  • package require shtmlview::shtmlview

Description

The shtmlview::doctools package extends shtmlview::shtmlview widgets with support for rendering of doctools-formatted documents.

Simply loading the package registers the support.

The package does not export any kind of command, class, etc.
